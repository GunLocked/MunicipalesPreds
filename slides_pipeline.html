Logique de génération — 40 docs par classe

for theme in THEMES:
  pool = PHRASES[label][theme]
  # tirage aléatoire SANS remise
  n = random.randint(3, 6)
  selected = random.sample(pool, n)
  doc[theme] = ' '.join(selected)
Chaque document = combinaison aléatoire de 3 à 6 phrases par thème.
random.sample garantit qu'une même phrase n'apparaît pas deux fois dans un doc.

Pourquoi TF-IDF ?

Valorise les mots rares dans le corpus global mais fréquents dans une classe.
Ex : « armement » → score élevé pour ED, faible ailleurs.
